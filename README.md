# 咨询师内容创作系统

当前版本：v1.4.0 公开源码非商业版。

给心理咨询师、人生教练、家庭教育指导师和其他助人服务者使用。

它不只是朋友圈写手。你可以把它理解成一个“个人 IP 与商业化行动搭档”：先判断你现在最该处理什么，再做出一个能拿去用的小结果，推动你发布、测试、询问或跟进，并根据真实反馈决定下一步。

## 它能做什么

1. **找定位**：从别人真实问过你的问题、你做过的判断和现有服务里，提炼值得测试的个人 IP 方向；
2. **找选题**：你没有知识库也能开始，它会从工作观察、常见误解、真实提问和个人经历里带你找第一条；
3. **多平台写作**：写朋友圈、小红书图文、短视频口播、公众号文章、服务介绍和活动说明；
4. **改稿**：缩短、重写、去 AI 味，把专业话和绕口话改成普通人第一遍就懂的话；
5. **商业化表达**：诊断为什么内容有人看却没人咨询，讲清服务适合谁、怎样进行、不能保证什么和下一步怎么了解；
6. **一稿多用与复盘**：把同一个核心判断分别改成不同平台内容，并根据真实发布反馈继续调整；
7. **安全检查**：提醒来访者隐私、未成年人、远程诊断、疗效承诺和危机内容风险。
8. **推动行动**：不让你一直讨论定位和改文案；每轮尽量给一个今天能完成、能获得真实反馈的动作。

## 它不是什么

- 不是只能写朋友圈的文案工具；
- 不是让你先填几十道题的定位课；
- 不是把术语堆得很专业的 AI 写手；
- 不是一次生成就能替代长期陪跑的“全自动商业顾问”；
- 不代替心理评估、治疗、督导、危机干预或法律意见。

## 它怎样帮助你往前走

每次尽量完成一个小闭环：

```text
判断当前卡点
→ 做出一份可用产物
→ 完成一个现实动作
→ 看一个关键反馈
→ 再决定下一步
```

例如，定位还不清时，它不会让你先填完一套定位表，也不会直接宣布“你就是某某教练”。它会先给一个测试方向，写出一条内容，让你发出去观察：来的是否是你想服务的人，他们记住了哪句判断，有没有人开始询问帮助。

免费系统会把这一小步完整做完，不故意藏关键方法。但个性化定位、完整产品设计、连续改稿、销售链路和长期行动督促，需要结合真实反馈多轮调整，不能假装一次对话就能替代真人咨询或陪跑。

## 朋友圈默认有多长

收到“写一条朋友圈”时，它默认写约 120—220 字，只讲一件事，不加标题，也不附创作分析。

- 你说“短一点”：约 60—120 字；
- 你说“写长一点”或指定约 300 字：约 250—450 字；
- 你明确指定字数时，以你的要求为准。

服务说明有时需要多讲一点，但它也不会把一条朋友圈写成一篇课程。

## 第一次怎么用

如果你不知道从哪里开始，直接说：

```text
咨询师内容创作系统，我第一次用。你能帮我做什么？
```

它会让你从眼前最急的一件事开始：定位没想清、完全不知道写什么、有素材不会写、已有稿子想改，或者内容没人咨询。

也可以直接下任务：

```text
我是家庭教育指导师，写一条关于家长越催、孩子越不动的朋友圈。写短一点，100 字左右。
```

```text
把这段工作观察改成一条小红书、一份 60 秒口播和一条朋友圈。三个版本不要只是长短不同。
```

```text
我写了很多拖延科普，收藏不少但没人咨询。先告诉我问题在哪里，再帮我改一条。
```

```text
我还没有明确定位。根据最近别人来问我的三类问题，帮我找一个值得测试的方向。
```

## 不需要知识库

本系统已经内置脱敏后的问诊方法、选题引导、商业原则、写作规则和通用示例。使用者不需要时澄的弹药库，也不需要先安装其他 Skill。

自己的历史内容、服务说明和客户常见问题可以让结果更像本人，但只是加分项，不是使用门槛。包里不含私人原文、客户资料、咨询记录、本地路径或弹药库结构。

## 安装

下载仓库 ZIP 后，把它交给 Codex，并说：

```text
请安装 ZIP 里的 skills/counselor-content-system。安装后告诉我是否需要新建任务。
```

## 使用边界

- 不上传完整咨询记录、会谈逐字稿和可识别聊天截图；
- 不根据一段网络描述给个人下诊断；
- 不承诺治愈、逆转或一定改变；
- 遇到迫在眉睫的自伤、伤人、虐待或急性风险时，先处理现实安全，不继续做内容。
