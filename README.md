# 咨询师内容创作系统

当前版本：v1.3.0 私人独立版。

给心理咨询师、人生教练、家庭教育指导师和其他助人服务者使用。

它不只是帮你写文案。它会先看你问的是不是对的问题。

比如你说“我不知道怎么卖自己”，它不会让你填个人品牌表。它会直接告诉你：客户不是来购买“你”，他是来判断你能不能帮他。然后从真实发生过的对话里，找出别人为什么会来找你。

## 这版解决了什么

- 不再拍脑袋给定位；
- 不把“温柔、犀利、真诚”当成服务价值；
- 一次只问一个真正有用的问题；
- 用户说“不知道”时，改问最近发生的具体事情；
- 先拆掉假问题，再给答案；
- 写朋友圈时直接给朋友圈，不附一堆创作术语；
- 把专业话改成普通人第一遍就懂的话；
- 内容需要商业作用，但不把“商业断点、价值翻译、可信支撑”说给用户听；
- 没有真实服务和反馈时，只给测试方向，不假装定位已经成立；
- 不靠真实来访者故事、诊断、恐惧和结果保证卖服务。
- 别人没有你的弹药库、知识库或其他 Skill，也能独立使用；
- 完全不知道写什么时，它会一次问一个小问题，最多三轮就做出测试稿；
- 它会识别“把话说复杂来证明专业”，先拆真问题，再改成人话；
- 它会把商业化理解为帮助客户做判断，不是硬塞预约；
- 没有真实服务时不装作能成交，先验证问题、判断和交付能力。

## 不需要知识库

这个 ZIP 已内置脱敏后的问诊方法、选题引导、商业原则、写作规则和通用示例。使用者不需要拥有时澄的弹药库，也不需要先整理自己的知识库。

自己的历史内容、服务说明和客户常见问题可以让结果更像本人，但只是加分项，不是使用门槛。包里不含私人原文、客户资料、咨询记录、本地路径或弹药库结构。

## 它怎么找定位

它只看三件已经发生的事：

1. 最近谁来找过你，对方原话怎么问；
2. 你说了什么，让对方换了个看法或定下一个动作；
3. 你现在到底能提供什么帮助，客户付钱后会经历什么。

这些证据不够时，它会明确说“这只是目前值得测试的方向”。

## 它怎么写内容

内部会判断这条内容说给谁、要说清哪件事、凭什么这样说、读者看完怎么办。

但这些框架不会出现在成稿里。用户拿到的应该是一篇能直接读、直接发的内容。

一旦某句话还需要解释，直接重写，不让读者猜。

## 可以直接这样用

```text
咨询师内容创作系统，我想做个人 IP，但完全没有概念。
```

```text
很多人和我聊完会说“哦，原来是这样”，但我不知道怎么介绍自己。
```

```text
我是家庭教育指导师，写一条关于家长越催孩子越不动的朋友圈。直接给成稿。
```

```text
这篇内容收藏不少，但没人来问。你先判断我真正写错了什么，再帮我改。
```

## 安装

把 ZIP 交给 Codex，并说：

```text
请安装 ZIP 里的 skills/counselor-content-system。安装后告诉我是否需要新建任务。
```

## 使用边界

这个 Skill 帮助内容创作和服务表达，不代替心理评估、治疗、督导、危机干预或法律意见。

- 不上传完整咨询记录、会谈逐字稿和可识别聊天截图；
- 不根据一段网络描述给个人下诊断；
- 不承诺治愈、逆转或一定改变；
- 遇到迫在眉睫的自伤、伤人、虐待或急性风险时，先处理现实安全，不继续做内容。
