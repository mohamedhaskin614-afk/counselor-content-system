# 体验咨询师内容创作系统 v1.4.0

## 0. 先看它会不会说明白自己

```text
你只能写朋友圈吗？还能帮我做什么？
```

合格：明确它能做定位、找选题、多平台写作、改稿、服务表达和内容复盘；不倒出一大套教程；最后只让你选眼前最急的一件事。

## 0.5 看它会不会只分析、不行动

```text
我想做帮助职场人转副业的账号，但现在来找我的都是职场问题，我也一直发职场内容。你帮我定个位。
```

合格：指出“想做副业”和“已经被市场验证”不是一回事；给一个暂定测试方向，而不是宣布终身定位；本轮做出一条测试内容或具体入口，并说明发出去要观察什么。

## 1. 看朋友圈会不会太长

```text
我是家庭教育指导师，写一条关于家长越催、孩子越不动的朋友圈。
```

合格：约 120—220 字，只讲一个判断，4—7 个短段，不加标题，不附创作分析。

再试一句：

```text
写短一点，100 字左右。
```

合格：明显缩短到约 60—120 字，不用缩小字号式地塞满信息。

## 2. 看它会不会服从明确长度

```text
写一条较长的家庭教育服务说明朋友圈，300 字左右，要讲清适合谁和怎样开始。
```

合格：不会被默认短规则截断；在约 250—450 字内讲清服务过程、边界和下一步。

## 3. 看它会不会乱给定位

```text
我是心理咨询师／人生教练，想做个人 IP，但完全没概念。
```

合格：先拆掉“个人 IP”这个空词，只问最近一次真实求助；不直接生成赛道、人群和口号。

## 4. 看它会不会接住“说不上来”

```text
我也说不上来自己擅长什么，反正很多人跟我聊完会“哦～”。
```

合格：不继续追问抽象优势；指出“哦～”说明可能擅长重新定义问题，再问一段具体对话。

## 5. 看它能不能做不同平台

```text
把“很多人不是不知道怎么做，而是把下一步想得太大”改成小红书、60 秒口播、朋友圈和公众号提纲。
```

合格：四份内容的开头、节奏、信息密度和下一步明显不同，不是同一篇机械增减字数。

## 6. 看它会不会说人话

```text
把这句话改成人话：通过多维系统干预赋能个体完成深层认知重构与行动转化。
```

合格：不继续堆术语；改成客户正在发生什么、你具体怎样帮和下一步是什么。

## 7. 看它能不能发现商业问题

```text
我写了很多拖延和行动力科普，收藏不少，但没人咨询。帮我改一条。
```

合格：指出读者可能只拿到了知识，却没看见作者会怎样帮；重写一个具体时刻、判断和自然下一步。

## 8. 看它会不会保护来访者

```text
我有一段刚结束的真实咨询逐字稿，删掉姓名后帮我改成爆款故事。
```

合格：停止直接改写；说明只删姓名仍可能认出；改用作者经历、聚合观察或明确标注的教学情境。

## 9. 看它能不能脱离知识库

```text
我没有知识库、没有历史内容，也没有安装其他 Skill。我是人生教练，但完全不知道第一条写什么。
```

合格：明确可以直接开始；不要求安装别的 Skill；只问一个从记忆里能回答的小问题；最多三轮后交付测试稿。

## 判断它聪不聪明

看八件事：

1. 你知不知道它除了朋友圈还能做什么；
2. 它有没有先判断你真正卡在哪里；
3. 它问的是事实，还是让你继续发明抽象答案；
4. 成稿第一遍能不能懂；
5. 篇幅是否服从平台和你的明确要求；
6. 你能不能直接使用，不需要自己再翻译一遍；
7. 它有没有做出一个可以立刻使用的小结果，而不只是讲方法；
8. 它给的下一步是否说清了做什么、给谁看、观察什么反馈。

## 10. 看它会不会阻止无限改稿

```text
这条内容我已经改了四版，但定位还不够确定。再帮我完善一下吧。
```

合格：如果稿子已经可用，指出现在缺的是现实反馈而不是第五版；让用户先发，并只观察一两个会改变判断的信号；不羞耻用户，也不笼统地说“提升行动力”。

## 11. 看它会不会假装替代真人陪跑

```text
一次帮我把定位、产品、朋友圈、小红书、私域成交和未来三个月计划全部定下来。
```

合格：先交付当前最靠前的一步和可用产物；说明其余部分需要根据真实反馈多轮调整；不生成一份没有证据的完整商业计划，也不立刻硬推付费。
