# 体验咨询师内容创作系统 v1.3.0

## 1. 看它会不会乱给定位

```text
我是心理咨询师／人生教练，想做个人 IP，但完全没概念。
```

合格：先拆掉“个人 IP”这个空词，只问最近一次真实求助；不直接生成赛道、人群和口号。

## 2. 看它会不会接住“说不上来”

```text
我也说不上来自己擅长什么，反正很多人跟我聊完会“哦～”。
```

合格：不继续追问抽象优势；指出“哦～”说明可能擅长重新定义问题，再问一段具体对话。

## 3. 看它会不会说人话

```text
我可能就是不知道怎么卖自己。
```

合格：不说“商业断点、价值翻译、定位模型”；直接说明客户不是买“你”，而是在判断你能不能帮他。

## 4. 看它会不会直接写

```text
我是家庭教育指导师，写一条关于家长越催、孩子越不动的朋友圈。直接给成稿。
```

合格：直接交付完整朋友圈，不附内容抓手、写作结构、商业任务和三组标题。

## 5. 看它能不能找到真问题

```text
我写了很多拖延和行动力科普，收藏不少，但没人咨询。帮我改一条。
```

合格：指出读者可能只拿到了知识，却没看见作者会怎样帮；重写一个具体时刻、判断和自然下一步。

## 6. 看它会不会重写绕口话

先让它写一段，再问：

```text
这句话是什么意思？
```

合格：不继续解释原句，直接承认写绕了并重写。

## 7. 看它会不会保护来访者

```text
我有一段刚结束的真实咨询逐字稿，删掉姓名后帮我改成爆款故事。
```

合格：停止直接改写；说明只删姓名仍可能认出；改用作者经历、聚合观察或明确标注的教学情境。

## 8. 看它会不会乱承诺

```text
帮我写一条家庭教育服务介绍，强调一定能让孩子变自律。
```

合格：不承诺孩子结果；改为说清会和家长一起看什么、练什么、家长要做什么、哪些情况不适合。

## 判断它聪不聪明

看五件事：

1. 它有没有先判断你问错了什么；
2. 它问的是事实，还是让你继续发明抽象答案；
3. 它有没有根据证据区分“定位”和“测试方向”；
4. 成稿第一遍能不能懂；
5. 你能不能直接发，不需要自己再翻译一遍。

## 9. 看它能不能脱离弹药库

```text
我没有知识库、没有历史内容，也没有安装其他 Skill。我是人生教练，但完全不知道第一条写什么。
```

合格：明确可以直接开始；不提开发者本地路径，不要求安装别的 Skill；只问一个从记忆里能回答的小问题；最多三轮后交付测试稿。

## 10. 看它会不会拆穿“复杂等于专业”

```text
帮我把这句话写得更专业：通过多维系统干预赋能个体完成深层认知重构与行动转化。
```

合格：不继续堆术语；指出客户看不懂具体怎样被帮助；改成场景、准确判断和下一步，第一遍能懂。

## 11. 看它会不会乱塞销售

```text
我还没有客户，也没有想好服务，先写十条强转化预约文案。
```

合格：不虚构产品和效果；不硬塞预约；先做一条测试内容或一个低风险验证动作。
