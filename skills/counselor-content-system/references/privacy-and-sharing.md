# 对外分享与脱敏

凡是产物要发到社交媒体、公开课、客户材料、GitHub 或其他任务之外，执行本检查。心理咨询、家庭教育与关系/生涯指导内容优先服从 `ethics-and-safety.md` 的更严格要求。

## 1. 先分类素材

- **公开**：作者本人已公开的信息、公开研究和公共资料；
- **经授权可分享**：有明确使用范围和期限的材料，但仍要判断权力差异与潜在伤害；
- **私人**：个人资料、未发布内容、内部流程、客户与来访者信息；
- **秘密**：凭据、Cookie、Token、密码、密钥和带权限链接。

只允许前两类进入公开产物。秘密直接移除，不在回复里复述。来访者材料即使已获授权，也不自动等于适合营销使用。

## 2. 扫描直接与间接标识

检查正文、元数据、文件名、截图、链接和图片，寻找：

- 姓名、昵称、头像、声音、手机号、邮箱、微信号和账号 ID；
- 城市、单位、学校、职位、罕见经历、家庭结构、精确日期与时间线；
- 诊断、健康、性、创伤、财务、法律、关系和未成年人信息；
- 客户名、订单、合同、群聊、咨询记录、测评、作业、反馈和后台数据；
- 本机路径、用户名、设备名、内部项目名和私人知识库结构；
- API key、Token、Cookie、密码、私有仓库和签名链接；
- 第三方完整文章、课程、量表、题库或其他受保护内容。

多个弱标识组合后仍可能识别人时，视为可识别。

## 3. 选择安全转写方式

优先顺序：

1. 删除真实故事，改写作者自己的经历；
2. 把多个观察抽象成聚合模式，不保留任何一人的故事线；
3. 使用明确标注的合成情境，只服务解释目的；
4. 仅在确有必要并完成专业伦理审查后使用真实材料的最小片段。

不要只替换姓名或改变年龄。不要为了「显得真实」保留无关细节。脱敏会破坏结论或仍可能识别时，暂停公开交付并让作者做专业判断。

## 4. 版权与证据

- 优先总结和独立表达，不复制第三方完整内容；
- 引用只取支持判断所需的短片段并标注来源；
- 没有许可的课程、文案库、量表、公式库和专属提示词不得打包公开；
- 合成案例、推断、匿名化内容和真实数据要分别标清；
- 不伪造客户评价、研究、数据或截图。

## 5. 交付前确认

公开产物必须满足：

- 不含可识别的来访者、客户、未成年人或普通用户信息；
- 不含本地路径、设备信息、凭据或带权限链接；
- 不含未经授权的业务数据和可反向识别的评论者信息；
- 不含大段受保护内容或未经许可的量表题目；
- 不把合成材料伪装成真实案例或反馈；
- 事实、推断、个人经验和专业建议边界清楚；
- 已再次执行 `ethics-and-safety.md` 的发布前红线检查。

发现并移除敏感信息后，只报告处理类别和方式，不重复原始秘密。
